# v2.0.4: ChatGPT 与豆包虚拟列表静默索引

## 主要修复

- ChatGPT 与豆包长对话不再依赖“滚完后一次读取 DOM”，改为按稳定消息 ID 缓存、去重并排序的消息索引。
- ChatGPT 优先读取当前会话的活动分支；接口不可用时仍可从已挂载 DOM 安全降级。
- 豆包移除自动遍历全页的行为，不会再陷入滚动循环或打断阅读位置。
- ChatGPT 目录跳转移除多余的自动滚动，只执行用户点击所需的一次定位。

## 交互改进

- 用户手动滚动，或点击目录后新消息挂载时，目录会静默补充新增内容。
- 豆包缓存按 `data-message-id` 去重，并按虚拟列表中的实际位置排序，避免目录重复或错序。
- 导出提示会明确标示内容是否来自被动索引，避免将当前缓存误称为已完整加载的 DOM。

## 安装测试

1. 下载 `ai-chat-export-pro-v2.0.4.zip`。
2. 解压到本地目录。
3. 打开 Chrome/Edge 扩展管理页面。
4. 开启开发者模式。
5. 选择“加载已解压的扩展程序”，加载解压后的目录。
