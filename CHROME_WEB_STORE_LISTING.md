# AI Chat Exporter Chrome 应用商店文案

> 用途：手动填写 Chrome Web Store 后台。代码块内为可直接复制的内容；说明文字不要复制到商店字段中。

## 商品名称

```text
AI Chat Exporter
```

品牌规则：Exporter 是产品名；Pro 只表示扩展内的付费功能，不再作为另一款插件名称。

## 简短说明

Chrome 应用商店的简短说明上限为 132 个字符。推荐使用：

```text
为 ChatGPT、DeepSeek、豆包等 AI 长对话生成侧边栏大纲，快速定位内容并导出 Markdown；Pro 支持勾选重要问答局部导出。
```

## 详细说明

```text
AI Chat Exporter 是一款面向 AI 长对话阅读、整理和本地保存的浏览器扩展。

打开受支持的 AI 对话页面后，扩展会在浏览器侧边栏生成问题与回答大纲。你可以点击标题快速定位内容，也可以将当前对话导出为 Markdown 文件，继续在 Obsidian、Notion、Typora 或其他知识管理工具中整理。

主要功能

• 侧边栏对话大纲：自动识别问题、回答和回答内标题
• 快速定位：点击大纲即可跳转到对应对话位置
• 阅读位置高亮：滚动长对话时同步显示当前阅读位置
• 完整 Markdown 导出：免费导出当前对话的已加载内容
• 格式保留：尽量保留正文、代码块、思考过程和搜索结果等可读取内容
• Pro 局部导出：勾选重要问题组，只导出所选问题及其完整答案

支持平台

DeepSeek、腾讯元宝、ChatGPT、豆包、Gemini、Grok、Kimi。

长对话加载说明

• ChatGPT：扩展会优先读取完整会话数据；如果页面暂时无法提供完整数据，则使用当前已加载的对话内容。
• 豆包：为避免扩展自动滚动页面并打断阅读，目录会随着你浏览对话逐步补全，导出以已经加载并完成索引的内容为准。

免费版与 Pro

免费版包含侧边栏大纲、快速定位、阅读位置高亮和完整 Markdown 导出。Pro 在免费功能基础上增加“勾选问题组局部导出”。免费基础功能不要求付费。

Pro 通过授权码解锁，无需注册账号、不绑定机器。购买会打开开发者提供的外部页面；开发者 Jeff 是 Pro 服务的销售方，Google 不参与交易。授权码属于数字商品，发放后不支持无理由退款；如遇重复付款或授权码无法激活，请联系开发者处理。

隐私说明

扩展仅在用户打开的受支持 AI 对话页面上读取内容，用于生成大纲和执行用户主动触发的导出。对话内容在浏览器本地处理，不上传到开发者服务器，不用于广告、分析或用户画像。扩展仅在本地保存界面状态和 Pro 激活状态。
```

## 单一用途说明

填写后台的“单一用途”或审核补充说明时使用：

```text
本扩展的唯一用途是帮助用户浏览、定位和导出其当前打开的 AI 对话内容。侧边栏大纲、阅读定位、完整导出和 Pro 局部导出都服务于同一用途：整理并保存 AI 对话。
```

## 权限用途说明

按后台实际出现的字段分别填写：

### sidePanel

```text
用于在浏览器侧边栏显示 AI 对话大纲、阅读位置和导出操作。
```

### activeTab

```text
仅在用户打开扩展时访问当前标签页，以识别当前 AI 对话并生成大纲或导出文件。
```

### scripting

```text
用于在受支持的 AI 对话页面中运行本地内容识别逻辑，提取用户当前可见或已加载的对话结构。
```

### tabs

```text
用于识别当前标签页是否属于受支持的 AI 对话平台，并在标签页切换时更新侧边栏内容。
```

### downloads

```text
用于在用户主动点击导出后，将生成的 Markdown 文件保存到用户本地设备。
```

### notifications

```text
用于向用户反馈导出完成或导出失败等操作结果，不用于广告或推广通知。
```

### storage

```text
用于在浏览器本地保存侧边栏界面状态、折叠状态和 Pro 授权激活状态，不保存对话正文。
```

### host permissions

```text
仅用于在用户访问受支持的 AI 对话平台时读取当前对话页面内容，从而生成侧边栏大纲并完成用户主动触发的 Markdown 导出。扩展不会读取其他网站，也不会将对话内容上传到开发者服务器。
```

## 隐私披露建议

后台的数据使用声明应与实际代码及隐私政策保持一致：

- 会处理“网站内容”，因为扩展需要读取 AI 对话正文。
- 会处理“网站活动”，范围仅限识别当前受支持网站和当前对话页面，目的为用户可见的大纲与导出功能。
- 数据只在本地处理，不出售、不用于广告、不用于信用判断，也不允许开发者人工读取。
- Pro 激活状态保存在本地；不要把授权码、付款凭据或用户付款信息写进公开说明。
