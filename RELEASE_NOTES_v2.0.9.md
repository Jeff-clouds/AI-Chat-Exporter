# AI Chat Exporter v2.0.9

## 本次更新

- 新增一次性、可关闭的首次使用提示：介绍目录定位和免费完整 Markdown 导出，不再用弹窗打断老用户。
- 侧边栏右上角新增“？”帮助入口，集中说明快速使用、ChatGPT/豆包长对话加载逻辑、免费版与 Pro、隐私说明。
- ChatGPT 显示“正在读取完整会话”；豆包明确提示目录会随用户滚动补全，扩展不会自动滚动页面。
- Pro 入口调整为“了解 Pro / 激活 Pro”，购买只在用户主动点击后打开。

## 免费功能不变

侧边栏大纲、快速定位和完整 Markdown 导出继续免费。Pro 仅提供勾选重要问题组的局部导出。

## 安装测试

1. 下载 `ai-chat-exporter-v2.0.9.zip` 并解压。
2. 打开 Chrome 或 Edge 的扩展管理页。
3. 开启“开发者模式”。
4. 选择“加载已解压的扩展程序”，加载解压后的目录。
