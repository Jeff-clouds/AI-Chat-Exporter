# AI Chat Exporter v2.0.8

## 本次更新

- 统一产品名称为 **AI Chat Exporter**。
- **Pro** 现在仅表示扩展内的付费功能层：可勾选重要问题组并进行局部导出；它不再是另一款独立插件。
- 同步更新 README、隐私政策和 Chrome 应用商店文案，使安装后名称、商店页面与 GitHub 资料保持一致。
- 修复发布打包脚本的版本读取逻辑，并统一发布包命名为 `ai-chat-exporter-v2.0.8.zip`。

## 功能说明

本次为品牌与发布流程更新，不改变免费功能边界：侧边栏大纲、快速定位和完整 Markdown 导出仍可免费使用。Pro 继续提供勾选问题组局部导出。

## 安装测试

1. 下载 `ai-chat-exporter-v2.0.8.zip` 并解压。
2. 打开 Chrome 或 Edge 的扩展管理页。
3. 开启“开发者模式”。
4. 选择“加载已解压的扩展程序”，加载解压后的目录。
