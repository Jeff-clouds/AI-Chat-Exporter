# v2.0.5: ChatGPT 回答大纲与侧栏排版修复

## ChatGPT 回答大纲

- 保留会话 API 的原始 Markdown 换行，回答中的 H1-H6 不再在索引阶段丢失。
- API 仍负责完整会话顺序；当前已挂载的 assistant DOM 标题会静默合并进对应回答，避免 API 成功时跳过真实页面标题。
- 修正 API 分支中 root 节点引入的 turn 编号偏移，使回答标题与 `conversation-turn-N` 对齐。
- 内容脚本更新后会替换页面中陈旧的消息索引实例，避免旧逻辑继续接管当前会话。

## 侧栏体验

- 问题前缀统一显示为 `Q1:`；ChatGPT 显示 `Q1: 你说：…`，豆包显示 `Q1: …`。
- 问题和回答目录统一使用覆盖中英文数字的字体，减少混排时的字距和基线不齐。

## 安装测试

1. 下载 `ai-chat-export-pro-v2.0.5.zip`。
2. 解压到本地目录。
3. 打开 Chrome/Edge 扩展管理页面。
4. 开启开发者模式。
5. 选择“加载已解压的扩展程序”，加载解压后的目录。
