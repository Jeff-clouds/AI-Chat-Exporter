# AI Chat Exporter v2.1.2 — ChatGPT 目录与性能紧急修复

## 修复内容

- 恢复 ChatGPT 的 API 目录读取，解决 v2.1.1 在部分当前页面上目录为空的问题。
- 侧边栏不再对 ChatGPT 执行任何全页消息节点、Prompt 或回答节点扫描。实际长对话页面上，这类查询可能阻塞页面。
- API 返回内容在页面内先压缩为当前会话分支的用户/助手消息，再交给扩展处理，减少跨上下文数据量。
- 侧边栏首次的重复目录请求会合并为一次。

## 更新后操作

更新扩展后，请刷新已经打开的 ChatGPT 标签页一次。若 API 暂时不可用，侧边栏会显示空目录，但不会再为生成目录而拖慢 ChatGPT 页面。
