# AI Chat Exporter v2.1.3 — ChatGPT 会话切换与目录稳定性更新

## 本次更新

- 每次目录请求都会绑定当前标签页地址和唯一请求标识。旧会话的迟到结果不会再覆盖当前会话目录。
- ChatGPT 在站内切换会话时，会立即清理上一会话的目录状态，并在已有提取任务结束后补做一次当前会话刷新。
- 会话数据桥接脚本从页面初始化阶段开始监听，通过路由代次隔离不同会话的请求和缓存。
- 目录与导出共用增量会话索引，减少对长对话页面的重复扫描。
- API 会话分支与当前已挂载回答标题按稳定消息 ID 合并，兼顾长对话完整性和页面内定位。
- 侧边栏会根据浏览器语言显示中文或英文，并补充更明确的长对话加载、导出和错误提示。

## 稳定性保护

- 新增会话切换隔离、性能回归、Markdown 标题、导出格式、目录索引和侧边栏交互测试。
- 所有对话内容仍在浏览器本地处理，不会上传到开发者服务器。

## 更新后操作

更新扩展后，请刷新已经打开的 ChatGPT 标签页一次，让新的页面桥接脚本从页面初始化阶段生效。
