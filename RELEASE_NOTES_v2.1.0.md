# AI Chat Exporter v2.1.0

## 性能修复

- 页面分析只在侧边栏打开时持续运行，关闭侧边栏后自动停止观察器、定时器、消息监听和豆包索引。
- 不再在插件安装、页面加载完成或后台切换标签时自动注入分析脚本。
- ChatGPT 完整会话请求按会话 ID 缓存并合并并发请求；最多保留最近 2 个会话，侧边栏关闭后释放缓存。
- 快速切换 ChatGPT 会话时，旧会话的迟到响应不会再进入当前目录；当前会话的新消息仍会增量补入。
- 豆包使用消息指纹进行增量转换，未变化的消息不再重复克隆和转换 Markdown。

## Pro 多格式导出

- 完整或局部对话支持 HTML、JSON、TXT。
- 完整 Markdown 导出继续免费。
- 局部导出以及 HTML、JSON、TXT 属于 Pro 功能，后台会独立执行权限检查。
- 已激活的旧 Pro 或 lifetime 授权自动获得新增格式，无需重新购买或补差价。

## ChatGPT 目录清理

- 从 ChatGPT 问题和回答目录中移除行首的“你说”“ChatGPT 说”“You said”“ChatGPT said”等角色标签。
- 不会误删正文中出现的相同文字，也不会改变豆包等其他平台内容。

## 安装测试

1. 下载 `ai-chat-exporter-v2.1.0.zip` 并解压。
2. 打开 Chrome 或 Edge 的扩展管理页。
3. 开启“开发者模式”。
4. 选择“加载已解压的扩展程序”，加载解压后的目录。
